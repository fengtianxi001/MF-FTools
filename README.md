## Project-Management-Tools

## 项目简介 

这是一个前端项目的管理工具,由于自己的项目分散在磁盘的各个文件夹内,每次检索和启动都费时费力,所以萌生出开发项目统一入口的想法.
	
## 功能
- 支持一键使用vscode打开项目
- 支持npm包一键install
- 支持一键打开文件夹
- 支持项目类型识别
- 支持文件夹名称搜索
- 自动扫描packahe.json中的script脚本,支持脚本的一键启动

## 快捷键
`ctrl + space` 快速激活和隐藏
